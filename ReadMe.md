# Golden Reports
## 목표

- 종류: 공부용 프로젝트
- 기간: 2026년 1월 24일 ~
- 위 프로젝트를 통해 CRUD 프로젝트의 복습 및 외부 API 호출 등에 대해 알 수 있었음
- 위 프로젝트를 통해 스프링 Security 내부에서 필터체인이 동작방식에 대해 알고, 커스텀 Security 필터를 사용해 JWT토큰 기반 인증 방식을 적용할 수 있었음
- 위 프로젝트를 통해 AuthenticationFilter -> AuthenticationManager -> AuthenticationProvider로 이어지는 폼로그인 및 세션 인증 기반 인증 방식에 대해서 학습할 수 있었음
- 추가적인 목표로 간단한 DB설계 방식과 최적화, 성능테스트의 기초에 대해 다룰 예정임- 
<br> <br> 
- 사용자의 활동을 토대로 사용자에게 적합한 종류의 영화/드라마 리뷰를 보여주는 기능입니다.
